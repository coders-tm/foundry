# Subscription Billing

## Summary
Expertise in subscription lifecycle management including plan management, feature usage, trial periods, auto-renewal, status transitions, and billing workflows.

## Key Domains
- Subscription plans and feature configuration
- Feature usage tracking and authorization
- Trial periods and conversion logic
- Auto-renewal and payment scheduling
- Subscription status lifecycle (active, paused, expired, canceled)
- Upgrade/downgrade workflows
- Grace period and cancellation handling

## When to Activate
- Creating or modifying subscriptions
- Checking user feature access via `HasFeature`
- Implementing trial-to-paid conversion
- Auto-renewal payment processing
- Subscription status transitions
- Feature usage metrics
