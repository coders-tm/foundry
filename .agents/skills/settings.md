# Settings

## Summary
Expertise in dynamic configuration management through the Settings model and ConfigLoader service, enabling runtime configuration changes without code deployment.

## Key Domains
- Setting key-value storage
- ConfigLoader config merging and caching
- Per-user or per-tenant scoped settings
- Setting groups and organization
- Feature flags and dynamic configuration
- Admin settings management
- Sensitive data handling

## When to Activate
- Reading or writing application settings
- Implementing admin configuration panels
- Dynamic feature flag management
- Per-tenant configuration
- Setting up application branding or theme
- Caching and performance optimization for config
