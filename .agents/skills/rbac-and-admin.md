# RBAC and Admin

## Summary
Expertise in role-based access control, permission hierarchies, admin authentication guards, and admin-specific workflows.

## Key Domains
- Module → Permission → Group hierarchy
- HasPermission trait and permission checking
- Admin guard separation from user guard
- Admin role definitions and scoping
- Permission validation in controllers and policies
- Admin action logging and audit trails

## When to Activate
- Implementing admin access control
- Checking user permissions via `HasPermission`
- Admin-only actions or routes
- Permission group management
- Admin authentication flows
- Admin dashboard features
