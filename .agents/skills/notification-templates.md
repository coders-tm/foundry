# Notification Templates

## Summary
Expertise in database-driven notification templates, template rendering, dynamic variable substitution, and notification dispatch.

## Key Domains
- Notification template storage and management
- Template variables and stub definitions
- NotificationTemplateRenderer service
- Multi-channel notifications (email, SMS, in-app)
- Template activation and distribution
- Language/localization support
- Event-triggered notifications

## When to Activate
- Sending system notifications
- Creating or modifying notification templates
- Rendering templates with dynamic data
- Multi-language notification support
- Notification template management in admin panel
- Event-driven notification dispatch
